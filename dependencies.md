# BDS_Console.py 依賴套件說明

本程式所需的 Python 套件如下：

## 必要第三方套件

| **套件** | **功能** | **建議版本** |
|---|---|---|
| **customtkinter** | 美化版 tkinter GUI 套件 | >= 5.2.0 |
| **requests** | HTTP 請求處理 | >= 2.31.0 |
| **schedule** | 排程任務管理 | >= 1.2.0 |

## Python 標準函式庫

|| **建議版本** |
|---|---|
| **Python** | >= 3.11.9 |


| **套件** | **功能** |
|---|---|
| **tkinter** | GUI |
| **subprocess** | 執行系統命令 |
| **threading** | 併發執行緒 |
| **json** | JSON 處理 |
| **os** | 作業系統介面 |
| **shutil** | 檔案操作 |
| **zipfile** | 壓縮檔處理 |
| **datetime, timedelta** (from **datetime**) | 日期時間處理 |
| **Path** (from **pathlib**) | 路徑處理 |
| **time** | 時間相關函式 |
| **sys** | 系統參數與控制 |
| **ctypes** | 呼叫 C 函式庫 |
